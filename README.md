# Pixel Perfect Portfolio

This is a responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

*   Single-page layout with smooth scrolling navigation.
*   Sections for Home, About, Skills, Projects, and Contact.
*   Animated skill bars.
*   Contact form with a success message.

## How to Use

To view the portfolio, simply open the `index.html` file in your favorite web browser.

The site is live at:https://coro326.github.io/portfolio-demo/
